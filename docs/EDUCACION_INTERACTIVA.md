# Educación interactiva

El módulo Educación dejó de ser una ventana con texto estático. Ahora funciona como un centro de aprendizaje dentro de la aplicación.

## Componentes

- rutas de aprendizaje por tema;
- búsqueda y filtros;
- tarjetas con progreso;
- lectura detallada en modal amplio;
- objetivos y listas prácticas;
- evaluación corta;
- resultado y retroalimentación;
- estado iniciado o completado;
- progreso por usuario.

## Persistencia

- `progreso_educacion`: avance y última lectura por usuario y artículo.
- `evaluaciones_educacion`: puntaje, total y respuestas.

Los datos se guardan localmente. Cada usuario del sistema puede construir su propio progreso sin depender de una cuenta en internet.

## Criterios de contenido

Una lección útil debe:

1. partir de una decisión real de finca, corte, beneficio, calidad o comercialización;
2. explicar el porqué, no solamente el procedimiento;
3. incluir una lista aplicable en campo;
4. diferenciar recomendaciones generales de requisitos legales o de certificación;
5. citar fuentes técnicas cuando corresponda;
6. evitar convertir una referencia demo en una prescripción agronómica universal.

## Cómo contribuir una lección

1. abra una propuesta en GitHub con objetivo, audiencia y fuente;
2. defina título, categoría, resumen, contenido y dificultad;
3. agregue una lista de comprobación;
4. proponga una pregunta de evaluación;
5. solicite revisión de una persona con experiencia cafetalera;
6. añada pruebas si cambia el comportamiento del módulo.
