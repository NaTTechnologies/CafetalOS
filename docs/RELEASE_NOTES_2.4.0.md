# Cafetal OS 2.4.0 — Planillas, acopio y productividad

## Novedades para cafetaleros

- Planilla semanal de cortadores con filas por persona y columnas por día/fecha.
- Temporadas de cosecha y valores predeterminados de pago, unidad y peso.
- Registro masivo para lotes, cortadores, cosecha, beneficio, inventario, gastos, proveedores, compras, clima y calidad.
- Pegado tabulado desde Excel y guardado atómico.
- Seguimiento estimado de costo y rentabilidad por semana de corte.
- Compras de café cereza, pergamino húmedo, pergamino seco, verde y tostado.
- Proveedores, recepción por peso, revisión de calidad e incorporación a inventario.
- Transformación de café comprado en beneficio.
- Educación con rutas, progreso y evaluaciones.
- Ayuda adaptable a pantallas pequeñas.
- Plantillas rápidas para datos institucionales y membrete PDF.

## Novedades para desarrolladores

- Nuevas tablas y migraciones idempotentes.
- Nuevos canales IPC y contrato preload.
- Nuevas tools MCP para planillas y compras.
- Pruebas de integridad demo, validaciones y flujos masivos.
- Capturas de compras, planilla y tablas masivas.
- Release multiplataforma automática al publicar la etiqueta `v2.4.0`.

## Compatibilidad

La aplicación migra bases anteriores al iniciar. Realice un respaldo antes de abrir una base productiva con esta versión. El lote técnico de acopio se oculta de la navegación y los KPI de finca.
